Api Hours

Api Hours est le module de gestion des périodes et horaires d'ouverture du système d'information touristique Apidae.

Ce formulaire peut être intégré au sein d'un formulaire web afin de permettre la saisie des informations d'ouverture. Les informations saisies sont ensuite stockées au sein d'une variable Javascript, dans le format attendu par l'API Apidae, ce qui permet de les transmettre plus facilement par la suite.
